## Data Structures Projects
## This repository contains a collection of C implementations for fundamental data structures and recursion exercises,
  organized into well-structured source, header, and test directories.

## Directory Structure:
src/ — All implementation files for data structures and recursion.

include/ — Header files corresponding to each module.

test/ — Test programs for validating implementations.

makefiles/ — Dedicated makefiles for building individual modules.

## How to Build
  Instead of relying on a single monolithic makefile, this project uses module-specific makefiles stored in the makefiles/ directory.
  This allows for more granular builds, useful when focusing on a specific topic or component.
